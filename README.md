# Krypto

A personal AI assistant built on a custom, continuously-evolving harness.

## What this is

Krypto is my own AI assistant — named after Superman's dog — running on a harness I build and own. Rather than relying on an off-the-shelf agent runtime, this project grows its own capabilities over time: new tools, integrations, and behaviors get added to the harness whenever a task calls for them.

## Intention

The goal is an assistant that becomes **self-improving**. Given its own machine, Krypto should eventually be able to:

- **Commit to itself** — make and ship changes to its own harness and codebase.
- **Add MCP connectors** — wire up new data sources and services on demand.
- **Download and install skills** — acquire new capabilities as needs arise.
- **Manage its own environment** — operate the machine it's deployed on.

## Approach

Iterative by design. The harness starts small and gains capabilities incrementally as real tasks reveal what's missing. Near-term work focuses on the foundations needed for safe self-modification; longer-term, the aim is for Krypto to extend itself with minimal hand-holding.

## Status

Early development — laying the groundwork.
